# Descrição

O objetivo deste desafio é criar uma aplicação que liste as infraestruturas de uma escola, 
permitindo a classificação de laboratórios ou salas de aulas como funcionais e não funcionais.

## Executar o projecto
```bash
$ ng serve
```
